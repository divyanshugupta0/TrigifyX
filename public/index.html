Install on your site

Paste this before the </body> tag on every page that has a form. The script captures all <form> submissions automatically.

The capture script is served from Netlify (24/7) and delivers submissions straight to Telegram — no backend server required. The bot token is injected from your deploy environment, so it's never in the source code.

How it works

  1. Generate

    Sign up and we issue a unique API key stored in your Firebase Realtime Database profile.

  2. Link Telegram

    Connect the Telegram account that should receive submissions via @TrigifyXbot.

  3. Embed

    Add the script snippet to your site. It auto-detects forms and captures submissions.

  4. Receive

    Each submission is sent through TrigifyXbot straight to your Telegram — instantly.
