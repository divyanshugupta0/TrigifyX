1. Getting Started

TrigifyX lets you receive form submissions instantly in your Telegram chat. No backend required — just paste one script tag.

  • Create an account on trigifyx
  • Link your Telegram account with @TrigifyXbot
  • Copy the install snippet from your dashboard
  • Paste it before </body> on every page with a form

2. How It Works

When a visitor submits a form on your site, TrigifyX captures the data and forwards it to your Telegram chat. The service includes:

  • Automatic form detection — works with any HTML <form>
  • Offline queue — submissions are retried if the network drops
  • Duplicate prevention — same submission is not sent twice
  • Origin protection — only registered sites can use your token
  • Rate limiting — prevents spam and abuse
  • Secure infrastructure — built with redundancy and hardened routing
  • Cloudflare-enabled security — DDoS protection, WAF, and secure access to the service
  • GDPR & Privacy — no data is stored on our servers
  • reCAPTCHA-enabled security — bot protection and verified authentication

4. Account & Security

  • Regenerate your access token anytime from the dashboard
  • Register all websites where you use the snippet
  • Enable Embed Gate in worker settings for stricter security
  • If your token is exposed, regenerate it immediately

5. Troubleshooting

  • Not receiving messages? Make sure you started a chat with @TrigifyXbot and copied the correct Chat ID.
  • Badge not showing? Check that the script tag is placed before </body> and that no ad blocker is interfering.
  • Token blocked? Your token may have been exposed from an unregistered origin. Regenerate it from the dashboard.
